**Google Merchandise Store Analysis (Power BI + SQL)**

** Objective**
Analyze user behavior, traffic sources, and revenue trends to identify key drivers of conversions and overall eCommerce performance.

**Tools Used**

* Power BI
* SQL (BigQuery dataset)
* Google Analytics Sample Data

Key Insights

* Significant revenue spike observed around mid-month, likely driven by promotions or high-value purchases
* Conversion rate remains relatively low (~1–2%), indicating opportunities to optimize the user journey
* High bounce rate (~48%) suggests potential UX or landing page inefficiencies
* Users with higher average order value (AOV) show stronger engagement and conversion behavior

---

## Business Impact

* Identified key periods driving revenue growth for better campaign planning
* Highlighted areas of user drop-off to improve conversion funnel
* Provided actionable insights for marketing optimization and user engagement strategies
